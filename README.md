# 🎵 Spotify Track Finder

A modern and responsive Spotify-inspired Track Finder built using **HTML, CSS, and JavaScript**. This project allows users to search for a song and view its details with a beautiful animated user interface.

---

## 🚀 Features

- 🎵 Search songs by name
- 🎨 Modern Glassmorphism UI
- 🔄 3D Flip Card Animation
- ⌨️ Typewriter Heading Animation
- 💚 Animated Spotify Theme
- ✨ Ripple Click Effect
- 🖱️ Interactive Hover Effects
- 📱 Responsive Design
- 🎧 Audio Preview (when available)
- 🔙 Search Again Feature

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/gauravchaturvedi727-oss/Spotify-track-finder.git
```

2. Open the project folder

```bash
cd Spotify-track-finder
```

3. Open `index.html` in your browser or use **Live Server** in VS Code.
